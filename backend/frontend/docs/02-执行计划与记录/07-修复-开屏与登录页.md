# 🔧 修复 6：开屏动画与登录页改造

**核心需求：**
1. 登录页删除重复品牌区域（AuthLayout 已有 logo+标题）

## 修改 1：路由守卫 — 所有人先看开屏

**文件：`src/router/index.js`**

- 根路由 `redirect` 从 `/home` 改为 `/splash`
- `beforeEach` 守卫顶部插入 splash 放行规则：
  - `to.path === '/splash'` → 始终 `return true`
  - 未登录访问需认证页面 → 跳 `/splash`（不再跳 `/auth`）
  - 已登录访问 `/auth` → 跳 `/home`（不变）

---

## 修改 2：SplashPage — 按 token 分流

**文件：`src/pages/auth/SplashPage.vue`**

动画结束回调 `zilingSplashFinished` 中读取 `localStorage.getItem('zl_token')`：
- 有 token → `router.replace('/home')`
- 无 token → `router.replace('/auth')`

其余结构不变（动态加载 CSS/JS、`#zilingSplash` 挂载、`z-index: 9999`、背景 `#FFFAF1`）。

---

## 修改 3：AuthPage — 删除重复品牌区域

**文件：`src/pages/auth/AuthPage.vue`**

⚠️ 只删模板，不动 `<script>` 和表单。

- **模板**：删除整个 `auth-header` div（含圆圈 SVG、"字灵"、`"AI 驱动的学习伙伴"`）
- **CSS**：删除 `.auth-header`、`.logo-container`、`.app-title`、`.app-subtitle` 四个样式块
- **CSS**：`.auth-page` 简化为基础居中容器（移除 `min-height: 100vh`、`background`、多余 padding，保留 `display: flex; align-items: center; justify-content: center`）

> 背景和 `max-width` 由 `AuthLayout.vue` 统一控制，AuthPage 只负责表单居中。
