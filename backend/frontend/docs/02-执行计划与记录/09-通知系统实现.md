### 4. `components/notifications/RingtonPickerSheet.vue` — 铃声选择弹窗

**去掉"分类映射"**，直接列出 `public/ringtones/alarm/` 下的 6 个文件：

```vue
<script setup>
// 直接使用文件名列表，无映射
const alarmFiles = [
  'facetalk_ringtone_incoming_default.mp3',
  'facetalk_ringtone_incoming_violin.mp3',
  'facetalk_ringtone_outgoing_default.mp3',
  'facetalk_ringtone_outgoing_violin.mp3',
  'voicetalk_ringtone_incoming_default.mp3',
  'voicetalk_ringtone_incoming_piano.mp3'
]

const props = defineProps({
  modelValue: String // 当前选中的文件名
})

const emit = defineEmits(['update:modelValue', 'close'])

const previewAudio = ref(null)

const selectRingtone = (file) => {
  emit('update:modelValue', file)
}

const previewPlay = (file) => {
  if (previewAudio.value) {
    previewAudio.value.pause()
  }
  previewAudio.value = new Audio(`/ringtones/alarm/${file}`)
  previewAudio.value.play()
}
</script>

<template>
  <div class="rington-picker-sheet">
    <div class="sheet-header">
      <span>选择闹钟铃声</span>
      <button @click="$emit('close')">确认</button>
    </div>
    <div class="sheet-body">
      <div 
        v-for="file in alarmFiles" 
        :key="file"
        class="rington-item"
        :class="{ 'is-active': modelValue === file }"
        @click="selectRingtone(file)"
      >
        <span>{{ file.replace('.mp3', '') }}</span>
        <button @click.stop="previewPlay(file)">▶ 预览</button>
      </div>
    </div>
  </div>
</template>
```

**样式要求**：
- 底部弹窗（fixed, bottom: 0, z-index: 1001）。
- 背景半透明遮罩（点击关闭）。
- 列表项高度 `44px`（符合移动端触摸标准）。
- 当前选中项：`background: var(--zl-primary-50)`（如果 token 存在，否则用 `--zl-bg-cool`）。
