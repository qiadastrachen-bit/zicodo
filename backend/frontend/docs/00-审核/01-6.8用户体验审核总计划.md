## ⚠️ 最高优先级硬性规则（4 条，违反任何一条立即停止）

1. **严格按照主提示词和设计规范**：所有 CSS 必须使用 `var(--zl-xxx)` design tokens，严禁硬编码颜色/字号/间距。除了 API 调用和交互逻辑的连接，**不允许破坏已有组件结构和页面架构**。

## Batch 0（前置任务）：全局品牌改名 —— 「字灵 / ZiLing」→「zicodo」

### 背景

旧版 APP 名称：**字灵**（中文）/ **ZiLing**（英文）
新版敲定名称：**zicodo**（全小写英文，无空格）

此任务在 **所有 Batch 之前完成**，确保后续修改都基于新名称。

### 改名原则（必须严格遵守）

| 类别 | 是否改名 | 说明 |
|------|---------|------|
| **用户可见文本**（模板中显示、页面标题、提示语、按钮文案） | ✅ 必须改 | 这是改名的核心目的 |
| **代码注释中的项目名** | ✅ 必须改 | `// 字灵聊天模块` → `// zicodo 聊天模块` |
| **文档 `.md` 文件中的项目名** | ✅ 必须改 | 标题、描述、示例中的「字灵」→「zicodo」 |
| **`index.html` 的 `<title>` 标签** | ✅ 必须改 | `<title>字灵</title>` → `<title>zicodo</title>` |
| **`package.json` 的 `name` 字段** | ✅ 必须改 | `"name": "ziling-frontend"` → `"name": "zicodo-frontend"` |
| **`public/manifest.json` 的 `name` / `short_name`** | ✅ 必须改 | PWA 安装后显示的应用名 |
| **localStorage key 前缀** | ✅ 必须改 | `ziling_chat_history` → `zicodo_chat_history`（同步修改读写代码） |
| **Store 中的常量字符串** | ✅ 必须改 | 如 `ziling_username` → `zicodo_username` |
| **文件系统目录名**（如 `Ziling-app/`、`ziling-backend/`） | ❌ 不改 | 目录名属文件系统结构，非用户可感知，改了会导致大量路径失效 |

function startGame() {
  gameState.value = 'playing'
  score.value = 0
  time.value = 0

  nextTick(() => {
    if (gameBoardRef.value) gameBoardRef.value.restart()
  })
}

function restartGame() {

function assignTaskToMember(teamId, taskData, memberId) {
  const team = teams.value.find(t => t.id === teamId)
  if (!team) return
  if (!isTeamLeader(teamId)) return  // 权限校验

  const member = team.members.find(m => m.id === memberId)
  const newTask = {
    id: 'team_task_' + Date.now(),
    title: taskData.title || '',
    description: taskData.description || '',
    assigneeId: memberId,
    assigneeName: member ? member.name : '未知',
    completed: false,
    createdAt: new Date().toISOString(),
    ...taskData
  }
  team.tasks.push(newTask)
  saveToStorage(teams.value)
  return newTask
}

<script setup>
import { ref, onMounted } from 'vue'
import { useRouter } from 'vue-router'
import ZlTopBar from '@/components/common/ZlTopBar.vue'
import EmptyState from '@/components/common/EmptyState.vue'

const router = useRouter()
const STORAGE_KEY = 'ziling_chat_history'
const dialogues = ref([])

// 从 localStorage 加载对话列表
onMounted(() => {
  try {
    const saved = localStorage.getItem(STORAGE_KEY)
    if (saved) {
      const history = JSON.parse(saved)
      // 按用户消息分割为多段对话
      const sessions = []
      let currentSession = null
      history.forEach(msg => {
        if (msg.role === 'user') {
          if (currentSession) sessions.push(currentSession)
          currentSession = {
            firstMessage: msg.content,
            firstReply: '',
            timestamp: Date.now(),
            messages: [msg]
          }
        } else if (currentSession) {
          if (!currentSession.firstReply && msg.type === 'quickReply') {
            currentSession.firstReply = msg.content
          }
          currentSession.messages.push(msg)
        }
      })
      if (currentSession) sessions.push(currentSession)
      dialogues.value = sessions.reverse()
    }
  } catch (e) { /* ignore */ }
})
</script>
