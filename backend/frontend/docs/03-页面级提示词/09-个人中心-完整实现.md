### 1. ProfilePage.vue — 一级主界面

**数据来源**：`userStore.user.nickname` + `petStore.petName` + 默认颜文字状态

**布局结构（从上到下）**：
```
[头像 88×88 圆形]          → default-avatar.png（项目固定默认头像，<img> 标签直显）

[o(*￣▽￣*)ブ]            → 颜文字状态文字，font-size: --zl-font-sm, color: --zl-text-hint

⬇ 间距 --zl-space-lg

┌─ 白色卡片（--zl-surface, --zl-radius-lg, --zl-shadow-sm）─┐
│ ZlListItem: icon=User          title="账户信息"    arrow    │
│ ZlListItem: icon=Palette       title="主题"        arrow    │
│ ZlListItem: icon=Sparkles      title="积分充值"    arrow    │
│ ZlListItem: icon=Settings      title="设置"        arrow    │
```

## 🔀 新增通用组件：ZlTopBar

**所有二级页面 共用** 这个返回栏组件，不要在每个页面手写。

**文件**：`components/common/ZlTopBar.vue`（新建）

**Props**：
- `title`（String, required）— 页面标题，居中显示
- `showBack`（Boolean, default true）— 是否显示返回箭头

**Events**：`@back` — 点击返回时触发，父组件里写 `router.back()`

**模板结构**：
```
┌─────────────────────────────┐
│  ← 返回      标题      占位  │  ← 右侧占位保持标题居中对称
└─────────────────────────────┘
```

**样式要求**：
- 高度：`44px`
- 背景：`var(--zl-bg-primary)`
- 标题：`font-size: var(--zl-font-lg)`，居中，`font-weight: 600`
- 返回箭头：`lucide-vue-next` 的 `ChevronLeft`，`18px`，左边 `--zl-space-md`
- 底部无分割线（极简风格）
- z-index：`100`（低于弹窗 1000，高于普通内容）

**使用方式**（以 ThemePage 为例）：
```vue
<ZlTopBar title="主题设置" @back="router.back()" />
```
