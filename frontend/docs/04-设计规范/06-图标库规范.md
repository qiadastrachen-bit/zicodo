# zicodo 图标库规范

## 概述

本文档定义了 zicodo 项目中图标使用的统一规范，包括图标选择、尺寸、颜色、交互状态等。

## 图标库

项目使用 **Lucide Vue Next** 作为图标库，这是一个现代化、开源的线性图标库。

- 官网: https://lucide.dev/
- GitHub: https://github.com/lucide-icons/lucide

## 图标使用规范

### 1. 引入方式

所有图标统一从 `lucide-vue-next` 引入：

```javascript
import { Sparkles, Plus, Settings, Home } from 'lucide-vue-next'
```

### 2. 尺寸规范

| 场景 | 尺寸 | 使用示例 |
|------|------|----------|
| 小按钮/列表项 | 16px | 标签、小型按钮 |
| 标准按钮/图标 | 18-20px | 导航栏、卡片图标 |
| 大图标/插图 | 24-32px | 引导页、空状态 |
| 超大图标 | 48px+ | 欢迎页、品牌展示 |

### 3. 颜色规范

| 状态 | 颜色变量 | 使用场景 |
|------|----------|----------|
| 主色 | `var(--zl-brand)` | 选中状态、强调图标 |
| 主要文本 | `var(--zl-text-primary)` | 按钮文字、标题图标 |
| 次要文本 | `var(--zl-text-secondary)` | 列表项图标、辅助图标 |
| 提示文本 | `var(--zl-text-hint)` | 占位图标、未选中状态 |
| 成功 | `var(--zl-success)` | 完成、正确状态 |
| 警告 | `var(--zl-warning)` | 警告、注意状态 |
| 错误 | `var(--zl-danger)` | 错误、删除状态 |

### 5. 交互状态

#### 悬停 (Hover)
```css
.icon:hover {
  color: var(--zl-brand);
  transform: scale(1.05);
}
```

#### 点击 (Active)
```css
.icon:active {
  transform: scale(0.95);
  opacity: 0.8;
}
```

#### 禁用 (Disabled)
```css
.icon:disabled,
.icon.disabled {
  color: var(--zl-text-hint);
  opacity: 0.4;
  cursor: not-allowed;
}
```

### 6. 组件封装示例

#### ZlIcon 组件（推荐）

```vue
<script setup>
/**
 * ZlIcon - 统一图标组件
 * Props:
 * - name: 图标名称 (Lucide 图标名)
 * - size: 尺寸 (default: 20)
 * - color: 颜色 (default: var(--zl-text-secondary))
 * - className: 自定义类名
 */
import { computed } from 'vue'
import * as LucideIcons from 'lucide-vue-next'

const props = defineProps({
  name: {
    type: String,
    required: true
  },
  size: {
    type: [Number, String],
    default: 20
  },
  color: {
    type: String,
    default: 'var(--zl-text-secondary)'
  },
  className: {
    type: String,
    default: ''
  }
})

const IconComponent = computed(() => {
  return LucideIcons[props.name] || null
})
</script>

<template>
  <component
    v-if="IconComponent"
    :is="IconComponent"
    :size="size"
    :color="color"
    :class="['zl-icon', className]"
  />
  <span v-else class="zl-icon-fallback">?</span>
</template>

<style scoped>
.zl-icon {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  transition: all var(--zl-transition-fast);
}

.zl-icon-fallback {
  width: var(--size, 20px);
  height: var(--size, 20px);
  display: inline-flex;
  align-items: center;
  justify-content: center;
  color: var(--zl-text-hint);
  font-size: var(--zl-font-sm);
}
</style>
```

### 7. 使用示例

#### 基础使用
```vue
<template>
  <div>
    <Plus :size="20" color="var(--zl-brand)" />
    <Settings :size="18" />
    <Sparkles :size="24" />
  </div>
</template>
```

#### 按钮中使用
```vue
<template>
  <button class="action-btn">
    <Plus :size="18" class="btn-icon" />
    <span>新对话</span>
  </button>
</template>

<style scoped>
.action-btn {
  display: flex;
  align-items: center;
  gap: var(--zl-space-sm);
}

.btn-icon {
  color: var(--zl-brand);
}
</style>
```

#### 列表项中使用
```vue
<template>
  <ZlListItem title="积分充值">
    <template #icon>
      <Sparkles :size="20" color="var(--zl-text-secondary)" />
    </template>
  </ZlListItem>
</template>
```

## 注意事项

1. **保持一致性**：同一功能始终使用相同图标
2. **语义化**：选择与功能语义相符的图标
3. **避免过度装饰**：图标应简洁明了，避免复杂装饰
4. **可访问性**：为图标添加适当的 `aria-label` 或 `title` 属性
5. **性能**：只引入需要的图标，避免全量引入
