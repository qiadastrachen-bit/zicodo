# 字灵(ZiLing) · 完整尺寸设计规范
# UI Designer 出品 — 基于 8pt Grid System
# 日期：2026-06-08

---

## 设计原则

1. **8pt Grid System**：所有尺寸均为 8 的倍数（8/16/24/32/40/48...）
2. **一致性**：相同功能的元素尺寸统一
3. **层次感**：通过尺寸差异建立视觉层次
4. **易用性**：可点击元素最小 44px（手指友好）
5. **品牌色不可更改**：`--zl-brand: #87C8B4`、`--zl-bg: #FFFAF1`

### 2. 间距系统（8pt Grid）

| Token | 数值 | 用途 | 示例 |
|-------|------|------|------|
| `--zl-space-0` | **0px** | 无间距 | 紧贴布局 |
| `--zl-space-1` | **4px** | 微间距 | 图标与文字间隙 |
| `--zl-space-2` | **8px** | 小间距 | 表单项内间距 |
| `--zl-space-3` | **12px** | 中间距 | 卡片内边距 |
| `--zl-space-4` | **16px** | 标准间距 | 页面左右边距、卡片间距 |
| `--zl-space-5` | **20px** | 大间距 | 段落间距 |
| `--zl-space-6` | **24px** | 区域间距 | 模块间距 |
| `--zl-space-8` | **32px** | 大区域间距 | 首页模块间距 |
| `--zl-space-10` | **40px** | 超大间距 | 页面顶部/底部留白 |
| `--zl-space-12` | **48px** | 分区间距 | 大模块分隔 |
| `--zl-space-16` | **64px** | 页面级间距 | 全屏布局分区 |

**代码实现**：
```css
:root {
  /* 间距 Scale（8pt Grid） */
  --zl-space-0: 0;
  --zl-space-1: 0.25rem;   /* 4px */
  --zl-space-2: 0.5rem;    /* 8px */
  --zl-space-3: 0.75rem;   /* 12px */
  --zl-space-4: 1rem;       /* 16px */
  --zl-space-5: 1.25rem;   /* 20px */
  --zl-space-6: 1.5rem;    /* 24px */
  --zl-space-8: 2rem;       /* 32px */
  --zl-space-10: 2.5rem;    /* 40px */
  --zl-space-12: 3rem;      /* 48px */
  --zl-space-16: 4rem;      /* 64px */
}
```

**使用规范**：
```css
/* ✅ 正确：使用 Token */
.card { padding: var(--zl-space-4); }
.page { padding: var(--zl-space-4) var(--zl-space-4); }

/* ❌ 错误：硬编码 */
.card { padding: 16px; }  /* 禁止使用！ */
```

### 7. 动画曲线（Animation Curve）

| Token | 数值 | 用途 | 效果 |
|-------|------|------|------|
| `--zl-transition-fast` | `150ms ease` | 快速反馈（按下、hover） | 即时响应 |
| `--zl-transition-normal` | `300ms cubic-bezier(0.4, 0, 0.2, 1)` | 标准动画（TabBar、弹窗） | iOS 标准曲线 |
| `--zl-transition-slow` | `500ms cubic-bezier(0.4, 0, 0.2, 1)` | 慢速动画（页面切换、引导） | 优雅过渡 |

**代码实现**：
```css
:root {
  /* 过渡动画 */
  --zl-transition-fast: 150ms ease;
  --zl-transition-normal: 300ms cubic-bezier(0.4, 0, 0.2, 1);
  --zl-transition-slow: 500ms cubic-bezier(0.4, 0, 0.2, 1);
}

/* 使用示例 */
.tab-item {
  transition: all var(--zl-transition-normal);
}

.btn:active {
  transition: transform var(--zl-transition-fast);
}
```

## ✅ 使用规范

### ✅ 必须这样做

1. **所有尺寸必须使用 Token**：
   ```css
   /* ✅ 正确 */
   .card { padding: var(--zl-space-4); }
   
   /* ❌ 错误 */
   .card { padding: 16px; }
   ```

2. **可点击元素最小 44px**：
   ```css
   /* ✅ 正确 */
   .btn-icon { width: 44px; height: 44px; }
   
   /* ❌ 错误（太小） */
   .btn-icon { width: 32px; height: 32px; }
   ```

3. **间距遵循 8pt Grid**：
   - 只用：`4/8/12/16/24/32/40/48/64px`
   - 不用：`5/10/13/17/...`（非 8 的倍数）

4. **动画使用标准曲线**：
   ```css
   /* ✅ 正确 */
   transition: all var(--zl-transition-normal);
   
   /* ❌ 错误 */
   transition: all 0.3s;  /* 硬编码 */
   ```
