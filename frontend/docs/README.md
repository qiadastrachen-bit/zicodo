## 📖 推荐阅读顺序

1. **01-项目总纲/** → 了解项目是什么、技术栈、核心决策
2. **00-审核/** → 了解当前已知问题和优先级
3. **04-设计规范/** → 了解设计令牌（颜色/字号/间距/动画）
4. **03-页面级提示词/** → 按页面查找具体开发指令
5. **02-执行计划与记录/** → 查看已执行的 Batches 和进度
6. **05-决策记录/** → 了解各模块的不可变决策（避免重复讨论）
