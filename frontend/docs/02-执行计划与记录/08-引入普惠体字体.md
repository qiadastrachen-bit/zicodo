## 📋 已确认的决定（不要再问用户）

### 1. 字体文件放在哪里？
**✅ 决定：选项A（`public/fonts/`）**

**理由**：
- Vite 自动处理：`public/` 目录下的文件会被原样复制到输出目录的根目录
- 访问路径简单：`/fonts/AlibabaPuHuiTi-2-55-Regular.woff2`
- 不需要额外配置：Vite 默认支持 `public/` 目录
- 构建速度更快：字体文件是静态资源，不需要被 Vite 处理

**目录结构**：
```
ziling-frontend/
├── public/
│   └── fonts/
│       ├── AlibabaPuHuiTi-2-55-Regular.woff2
│       ├── AlibabaPuHuiTi-2-65-Medium.woff2
│       └── AlibabaPuHuiTi-2-75-Bold.woff2
└── ...
```

### 5. 需要支持哪些字重？
**✅ 决定：选项A（Regular (400) + Medium (500) + Bold (700)）**

**理由**：
- 设计规范（`design-system-spec.md`）中已经定义了这个方案
- 足够使用：
  - Regular (400)：正文
  - Medium (500)：按钮、导航、卡片标题
  - Bold (700)：页面标题、强调文字
- Light (300) 不适合移动端：
  - 在手机小屏幕上偏细，长时间阅读容易累
  - 学生用户可能在公交、教室等**光线复杂环境**使用，需要足够清晰度
  - 400 已经是很好的正文选择，300 性价比不高

**对应的 CSS 定义**：
```css
--zl-weight-regular: 400;   /* 正文 */
--zl-weight-medium: 500;    /* 按钮、导航 */
--zl-weight-bold: 700;       /* 标题 */
```

**不要加 Light (300)**，除非有特殊需求（比如辅助文字、水印等）。

### Step 2：转换为 WOFF2 格式（如果下载的是 TTF）

#### 2.1 为什么需要转换？

- TTF 格式文件较大（每个约 3-5MB）
- WOFF2 格式文件较小（压缩率约 40%，每个约 1-2MB）
- WOFF2 是 Web 专用格式，加载更快

#### 2.2 转换工具（任选一个）

**在线工具**：https://cloudconvert.com/ttf-to-woff2

**本地工具**：`woff2` 命令行工具

**安装 woff2**：
```bash
npm install -g woff2
```

**转换命令**：
```bash
ttf2woff2 < public/fonts/AlibabaPuHuiTi-2-55-Regular.ttf > public/fonts/AlibabaPuHuiTi-2-55-Regular.woff2
ttf2woff2 < public/fonts/AlibabaPuHuiTi-2-65-Medium.ttf > public/fonts/AlibabaPuHuiTi-2-65-Medium.woff2
ttf2woff2 < public/fonts/AlibabaPuHuiTi-2-75-Bold.ttf > public/fonts/AlibabaPuHuiTi-2-75-Bold.woff2
```

#### 2.3 如果无法转换

**临时方案**：先使用 TTF 格式，部署前再转换。

**修改 `@font-face` 定义**：
```css
@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.ttf') format('truetype');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

### Step 3：在 `tokens.css` 中引入字体

#### 3.1 编辑 `src/styles/tokens.css`

**在 `:root` 前面添加 `@font-face` 定义**（不能放在 `:root` 里面）：

```css
/* ===================================================================
   0. 字体引入（阿里巴巴普惠体）
   =================================================================== */

@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;  /* 使用 swap，避免字体加载期间文字不可见 */
}

@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-65-Medium.woff2') format('woff2');
  font-weight: 500;
  font-style: normal;
  font-display: swap;
}

@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-75-Bold.woff2') format('woff2');
  font-weight: 700;
  font-style: normal;
  font-display: swap;
}

:root {
  /* ===================================================================
     1. 颜色系统
     =================================================================== */
  --zl-brand: #87C8B4;
  /* ... 保留已有颜色 ... */
  
  /* ===================================================================
     2. 字体系统（7级层次）
     =================================================================== */
  --zl-font-family: 'Alibaba PuHui Ti', -apple-system, BlinkMacSystemFont, 'Noto Sans SC', sans-serif;
  --zl-font-xs: 12px;
  --zl-font-sm: 14px;
  --zl-font-base: 16px;
  --zl-font-md: 20px;
  --zl-font-lg: 24px;
  --zl-font-xl: 30px;
  --zl-font-xxl: 36px;
  
  --zl-weight-regular: 400;
  --zl-weight-medium: 500;
  --zl-weight-bold: 700;
  
  /* ... 其他 Token ... */
}
```

**注意事项**：
1. `@font-face` 必须放在 `:root` 外面（不能嵌套）
2. `font-display: swap` 很重要，避免字体加载期间文字不可见
3. 备用字体（`-apple-system` 等）必须保留，防止字体加载失败

## ⚠️ 注意事项

### 1. 不要删除备用字体

```css
/* ✅ 正确：保留备用字体 */
--zl-font-family: 'Alibaba PuHui Ti', -apple-system, BlinkMacSystemFont, 'Noto Sans SC', sans-serif;

/* ❌ 错误：删除备用字体 */
--zl-font-family: 'Alibaba PuHui Ti';  /* 字体加载失败时会显示默认字体，不美观 */
```

---

### 2. 必须使用 `font-display: swap`

```css
/* ✅ 正确：使用 swap */
@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;  /* ← 必须加这个 */
}

/* ❌ 错误：没有 font-display */
@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  /* 缺少 font-display，字体加载期间文字不可见 */
}
```

---

### 3. `@font-face` 不能放在 `:root` 里面

```css
/* ✅ 正确：@font-face 放在 :root 外面 */
@font-face {
  font-family: 'Alibaba PuHui Ti';
  src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}

:root {
  --zl-font-family: 'Alibaba PuHui Ti', sans-serif;
}

/* ❌ 错误：@font-face 放在 :root 里面 */
:root {
  @font-face {  /* ← 这是错误的！ */
    font-family: 'Alibaba PuHui Ti';
    src: url('/fonts/AlibabaPuHuiTi-2-55-Regular.woff2') format('woff2');
  }
  
  --zl-font-family: 'Alibaba PuHui Ti', sans-serif;
}
```
